# copy paste in vim happens in normal mode
1. v        - select characters
    shift-v  - select whole lines
    Ctrl-v   - select rectangular blocks 
        Ctrl-q   - if Ctrl-v is mapped to paste

2. navigate
3. cut/copy
   d - cut
   y - copy (yank)
4. navigate
5. paste
    Shift-p - paste before cursor
    p - paste after cursor
